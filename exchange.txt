## Binance Exchange


-> Binance Exchange Web tarafında ; 

* Node.Js 
* React.Js
* Redux
* jQuery
* Bootstrap
* Sass
* Webpack
* MongoDB

Teknolojilerini base teknoloji olarak kullanmıştır. Service kullanımlarını C++ ve Go ile yazmıştır.

-> Binance Exchange Mobile tarafında ;

* Kotlin
* Swift

Teknolojilerini kullanıp web bağlantılarına göre işlemlerini gerçekleştirmektedir.
